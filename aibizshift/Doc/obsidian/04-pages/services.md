### Offres principales (4 grosses cards)

#### 1. Consulting IA & Automatisation
- Promesse : gagner 10-20h/semaine via workflows.
- Livrables : audit maturité IA, workflows n8n (jusqu'à 40+/projet), intégrations API (CRM, facturation, emailing), doc + formation.
- Prix : **à partir de 500€/jour**.

#### 2. Création & Refonte de Sites Web IA
- Promesse : site qui convertit, hébergement France.
- Livrables : audit marketing 14 sections, site + blog + admin, SEO on-page et technique, hébergement souverain RGPD.
- Prix : **à partir de 1 500€**.

#### 3. Formation IA & Automatisation
- Promesse : 4 modules progressifs, direct ou via Qualiopi.
- Modules :
  - **M1 — Comprendre l'IA** · 1/2 journée · Tous publics, zéro prérequis.
  - **M2 — Automatiser avec l'IA** · 1 jour · n8n, Airtable (intermédiaire).
  - **M3 — Créer un agent IA** · 1 jour · no-code (intermédiaire+).
  - **M4 — Vibecoding** · 2 jours · Claude Code (avancé).

#### 4. Développement SaaS sur mesure
- Promesse : application web complète en vibecoding sous supervision AMOA.
- Livrables : spec fonctionnelle, frontend+backend+BDD, panel admin, déploiement souverain.
- Prix : **à partir de 5 000€** — forfait projet.
